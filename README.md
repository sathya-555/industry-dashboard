# Industry-Oriented Web Dashboard for Data Visualization

## Project Overview
This project is a **responsive, interactive web dashboard** designed to provide real-time insights into key business metrics such as **sales, revenue, user activity, and active projects**. It is developed using **HTML, Tailwind CSS, JavaScript, and Chart.js**, all in a single file for simplicity.

Businesses often need quick visibility into performance data, but static pages fail to present insights effectively. This dashboard addresses that need by providing **dynamic charts, live counters, interactive filters, and a professional aesthetic** suitable for industry applications.

---

## Features

- **Live Counters:** Total Sales, Revenue, New Users, and Active Projects update automatically to simulate real-time data.
- **Interactive Chart:** Monthly revenue is displayed in a dynamic **line chart** with updates every few seconds.
- **Filterable Table:** Recent transactions can be filtered by category (Sales, Marketing, Finance).
- **Responsive Design:** Works seamlessly on **desktop, tablet, and mobile devices**.
- **Dark Mode Toggle:** Users can switch between light and dark themes for a modern look.
- **Portfolio-Ready:** Clean design and professional layout suitable for web developer portfolios.

## How to Use

1. Open `index.html` in a browser.
2. Use the **Filter Transactions** dropdown to filter the table.
3. Click **Toggle Dark Mode** to switch between light and dark themes.
4. Observe the **live counters** and **updating revenue chart**.
